Пример:
От Прямоугольника наследуется Квадрат. У обоих есть методы вычисления периметра и площади.

У Прямоугольника:
Предусловие: длина и ширина положительны.
Постусловие: Метод calculateArea возвращает положительное значение (ширина * длина).
Постусловие: Метод calculatePerimeter возвращает положительное значение (2 * (ширина + высота)).

У Квадрата:
Предусловие: положительна только длина стороны (ослабление предусловия).
Постусловие: Метод calculateArea возвращает положительное значение (длина ^ 2).
Постусловие: Метод calculatePerimeter возвращает положительное значение (4 * длина).
Постусловие: Метод getLength возвращает значение длины стороны на основании площади квадрата
(корень из площади). Длина стороны должна равняться изначальной длине (усиление постусловия).
